# Tasbeh Counter

A simple digital Tasbeh (prayer bead) counter built with HTML, CSS, and JavaScript. Keep track of your dhikr and tasbeeh easily.

## Demo
[Live Demo](https://tasbehcounter.vercel.app/)

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Click button to increment counter
- Reset counter anytime
- Minimalist and responsive design
- Tracks dhikr counts digitally
